# AceCG Documentation

This directory contains public-facing AceCG documentation.

Recommended reading order:

1. `developer_guide/00_architecture.md`
2. `developer_guide/03_compute.md`
3. `developer_guide/04_mpi_runtime.md`
4. `developer_guide/08_schedulers.md`
5. `developer_guide/09_workflows.md`

The developer guide describes the current package architecture, runtime
contracts, scheduler behavior, workflow semantics, and I/O conventions. Config
examples live under `../configs/templates/`.
